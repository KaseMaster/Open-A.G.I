### Anomaly Detection Algorithm
The observer agent implements a robust statistical approach to anomaly detection:

1. **Running Statistics**: Maintains mean and standard deviation for monitored fields
2. **Z-Score Calculation**: Computes deviation from normal behavior
3. **Threshold-Based Detection**: Configurable sensitivity (default: 3.0σ)
4. **Severity Classification**: Low/Medium/High anomaly categorization
5. **Trend Analysis**: Long-term behavior monitoring
