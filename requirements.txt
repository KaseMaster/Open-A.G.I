# INSTALACIÓN SELECTIVA
# =====================
# 
# Para instalar solo las dependencias core:
#   pip install cryptography pycryptodome pyotp aiohttp websockets pydantic python-dotenv click rich tqdm
#
# Para instalar con testing:
#   pip install -r requirements.txt
#   # O específicamente las de testing: pip install pytest pytest-asyncio pytest-cov pytest-html
#
# Para instalar almacenamiento:
#   pip install aiosqlite redis aioredis
#
# Para instalar monitoreo:
#   pip install loguru prometheus-client psutil pandas flask Flask-SocketIO
#
# NOTAS IMPORTANTES
# =================
# 
# 1. El framework está diseñado para funcionar con dependencias opcionales
# 2. Si una dependencia no está disponible, la funcionalidad relacionada se deshabilita automáticamente
# 3. Los tests se omiten (skip) si las dependencias requeridas no están presentes
# 4. Todas las importaciones son condicionales y manejan ImportError apropiadamente
# 
# COMPATIBILIDAD
# ==============
# 
# - Python 3.8+
# - Windows, Linux, macOS
# - Arquitecturas x86_64 y ARM64
