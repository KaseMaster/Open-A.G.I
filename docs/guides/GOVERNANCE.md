# Harmonic Currency Governance Charter

## Purpose  
Build an open, transparent, coherence-based financial protocol that rewards ethical energy contribution, not extraction.

## Principles
1. Transparency and reproducibility.
2. Consent in data collection.
3. Non-violence — no weaponized or coercive use.
4. Decentralized validation by measurable coherence.
5. Open participation; one node one vote.

## Decision Process
- Parameter changes (e.g., mint_threshold) by 2/3 validator majority.
- Emergency safety shutdown by ≥60 % quorum.

## Values
Human liberation, ecological balance, open knowledge.
